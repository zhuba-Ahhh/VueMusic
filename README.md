# VueMusic

## 🌈介绍

该项目是基于 Vue 和 ElementUI 组件库 搭建的一个PC端的音乐网站，目前Vue2版本已经基本完成，需要进一步优化性能，改善体验,Vue3版本正在计划中。

### [Vue2](./Vue2/README.md)
### Vue3 待完成ing
